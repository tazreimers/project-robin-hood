# Safety And Compliance

Project Robin Hood provides analytics and manual bet instructions only.

Do not implement:

- Automatic betting.
- Bookmaker login automation.
- Bookmaker account control.
- Captcha bypassing.
- KYC bypassing.
- Geolocation bypassing.
- Age-check bypassing.
- Self-exclusion bypassing.
- Responsible gambling control bypassing.
- Anti-bot bypassing.
- Scraping that violates site terms.

Prefer official APIs and permitted data feeds. Before adding a provider, check legal requirements, API terms, bookmaker terms, and responsible gambling obligations in the relevant jurisdiction.

Manual execution records are user-entered notes. They must not become automated account actions.
